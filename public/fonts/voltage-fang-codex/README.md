## Limits

If the static font was built with the open-font remix engine, it is an OFL derivative with source license included in `static/OFL.txt`. The package is not trained on arbitrary font downloads. The static build has no manual kerning pass, GSUB alternates, hinting, or final variable binary. Use the proofs to art-direct the next pass.
