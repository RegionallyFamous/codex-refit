Taste Decisions

  • Weight is intentionally loud at 125.0 units, so the face reads as title/display rather than body text.
  • Slant is set to -16.1 degrees for forward motion.
  • Spike energy 0.29 and jitter 0.02 create controlled weirdness while preserving alphabet recognition.
  • Spacing is conservative before kerning; this generator prioritizes usable proofs over tight logo-only spectacle.
  • The workflow avoids arbitrary font scraping and records prompt, seed, source font, license, and deterministic transforms for provenance.
  • Every uppercase and numeral has a recorded construction idea; Codex should reject the result if those ideas are not visible in the raw black proof.
  • The fused-outline engine uses single silhouette contours and reversed counter contours before decorative color treatment.
